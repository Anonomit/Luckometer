1.4.0

Features

  • Maximum storage limits can now be set in the options menu.
  • Added a filter for lucky items, which tracks whether you had certain lucky items on you when you won a roll.

Changes

  • Luck is now calculated from the p-value of your rolls against the expected average. In practice, luck is no longer statistically guaranteed to approach 50% with enough rolls.
  • Improved the appearance of the stats window when no rolls are in the database.

Fixes

  • Character metadata in database is now updated when logging in, just in case of race or name change.
  • Optimization fixes. The options menu should be more responsible about how often it refreshes.

 

1.3.0

Features

  • Started tracking whether the player is carrying certain lucky items with them. No way to filter roll stats for this yet.
  • Added some more filters.

Changes

  • Database improvements. Roll counts by character are now shown.

Fixes

  • Fixed an issue where roll filtering would never complete if options were changed too quickly.
  • Incomplete rolls that started during login or before a UI reload should now be correctly stored once they end.
